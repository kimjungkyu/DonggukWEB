- A Domestic crime Using inquiry system -

Difference between existing research and this study
  Existing research
    1. Use of public data.
    2. There are not enough functions for necessary searches such as year, region, etc.
    3. The function of the graph is inconvenient because of the large number of data.
  This study
    1. Use existing public data for the [2015-2016] year.
    2. Ensure that users who want to identify crimes such as year, region, and city can search.
    3. Use the Google API to add graphs and divide by year to show local data for 2015 and 2016.
  
Service function
    1. Start index.jsp with non-member services, member services, and administrative services.
    2. Additional features include advertisements, graphs, table tables, subscriptions, and login services.
    
Future plans
    1. We will collect the numerical information of domestic criminals to be continuously updated and apply the Spring Framework.

[2015-2016] 국내 지역 범죄율 조회 시스템

기존 연구 및 본 연구의 차이  
  기존연구
    1. 공공데이터의 사용한다.
    2. 연도, 지역등 필요한 검색에 대한 기능이 미비하다.
    3. 그래프의 기능이 데이터의 수치가 많아 보기 불편하다.
  본 연구
    1. 기존 [2015-2016] 연도의 공공데이터의 사용한다.
    2. 연도, 지역, 시 등의 범죄를 확인하고 싶은 사용자가 검색을 할 수 있도록 한다.
    3. 구글API를 사용하여 그래프를 추가하고 년도별로 나누어 2015, 2016년도의 국내 지역 데이터 수치를 보여주도록 한다.
  
서비스기능
    1. 비회원 서비스, 회원 서비스, 관리자 서비스로 index.jsp (을/를) 시작한다.
    2. 추가사항으로는 광고, 그래프, 테이블 표, 회원가입, 로그인 서비스를 구현한다.
    
추후 계획
    1. 지속적으로 갱신될 국내 범죄자 수치 정보들을 수집하고 Spring Framework를 적용 시킬 것이다.
